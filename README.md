A Trello-like web app to manage projects, boards, lists, and tasks. Users can organize their work visually, drag-and-drop tasks between lists, and track progress efficiently. Built with React and Tailwind CSS, this app focuses on a clean, responsive interface with offline data persistence.

Features
Boards: Create multiple boards for different projects.
Lists: Organize tasks into lists such as "To Do", "In Progress", "Done".
Cards/Tasks: Add tasks to lists, edit, or delete them.
Drag-and-Drop: Move tasks between lists easily.
Modals: Create and edit boards, lists, and tasks via pop-ups.
Responsive Design: Works on desktop and mobile devices.
Dark/Light Mode: Toggle between themes.
